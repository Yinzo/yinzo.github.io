新建VPS应该做的事情

  1. 创建root密码

    passwd
    1. 安装oh-my-zsh，这里是debian， 所以使用apt-get bash apt-get update; apt-get upgrade

  2. 安装zsh、git、pip

    sudo apt-get install zsh git python-pip python-dev
    1. 安装oh my zsh bash wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh

  3. 更换默认shell为zsh

    chsh -s /usr/bin/zsh

    这个路径不一定，如果不知道，可以使用which zsh来查看zsh位置

    1. 上传.zshrc bash scp ~/.zshrc root@*.*.*.*:~/ 此处*为ip地址

  4. 然后就要修改.zshrc了

    1. 改zsh对应路径
    2. 还原插件列表
    3. 检查个性化设置如alias

  5. 安装shadowsocks

    pip install shadowsocks
    1. 建立 /etc/shadowsocks.json
    2. 在 /etc/rc.local 中加入shadowsocks的开启自启动 bash /usr/bin/python /usr/local/bin/ssserver -c /etc/shadowsocks.json -d start
