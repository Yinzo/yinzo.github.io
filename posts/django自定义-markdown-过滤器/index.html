Django自定义 Markdown 过滤器

Django自定义 Markdown 过滤器

2014/12/15 14:5 下午
1. 在你 APP 目录下建立tamplatetags文件夹，手动建。文件夹和 APP 的 view.py 处于同一级 2. 手动新建文件 __init__.py ，置空就好了，用于 Django 识别。 3. 新建单独的 py 文件，文件名为过滤器调用时的名字，在其中输入你过滤器的代码。这里我使用的文件名是 my_markdown.py ，具体代码下附。 4. 在模板文件中载入过滤器使用。{% load my_markdown %}

附 my_markdown.py 代码

from django import template
from markdown import markdown

register = template.Library()

@register.filter(name='my_markdown')
def my_markdown(value):
        return markdown(value)
